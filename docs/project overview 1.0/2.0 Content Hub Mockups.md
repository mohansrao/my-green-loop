## Responsive Breakpoints

- **Mobile**: < 768px - Single column layout
- **Tablet**: 768px - 1024px - 2 column layout  
- **Desktop**: > 1024px - 3 column layout

## Design Principles

✅ **Clean & Minimal** - Focus on visual content with minimal text
✅ **Touch-Optimized** - Adequate spacing and target sizes on mobile/tablet
✅ **Consistent Branding** - Eco-friendly green accents throughout
✅ **Fast Scanning** - Image-first cards with clear typography hierarchy
✅ **Progressive Disclosure** - Descriptions/tags shown on hover/click, not on cards
